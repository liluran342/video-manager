# 🎬 Video Manager（本地视频管理系统）

一个基于 Node.js + Express 的本地视频管理工具，类似“私人版 Netflix”。

## 🚀 功能

- 📂 自动读取本地视频文件
- 🎥 点击播放视频
- 🖼️ 视频列表展示
- ⚡ 轻量简单，无数据库

## 📁 项目结构
video-manager/
├── public/ # 前端页面
├── videos/ # 本地视频目录
├── server.js # 后端服务
├── README.md
